ANALYSIS RULES:
1. MODE ADAPTATION:
   - "joined": Patient + encounter + diagnosis data combined is the primary truth.
   - "both": Start population-level (joined), then drill into condition/provider nuances.
   - "separate": Find patterns across departments, facilities, or diagnostic categories.

2. DATA INTEGRITY:
   - NO IDs: Never reference patient IDs, provider IDs, or claim IDs as analytical dimensions.
   - DISCRETE: Treat diagnosis codes, procedure codes, and severity tiers as categorical.
   - SIGNIFICANCE: Correlations < |0.10| = "Noise". Always report N for clinical subgroups.
   - QUALITY: Apply SEMANTIC NULLS PROTOCOL (rule 6) — distinguish mandatory clinical fields from sparse event columns.
   - CAUSATION: Healthcare data is observational. NEVER imply causation without strong correlation (>0.7).
     Always use "associated with", never "causes".

3. DATA HONESTY: No speculation. Clinical claims require strong evidence.
   For mandatory clinical field nulls >5% (e.g., Age, Diagnosis Code, Admission Date), you MUST add a "Data Limitations" subsection.
   For sparse clinical event columns (e.g., Readmission, Complication), analyze the non-null "High-Risk Cohort" — do NOT add a limitations subsection.
   If N<50 for a subgroup, explicitly label as "preliminary / low confidence".

4. STRUCTURE:
   - Link patient demographics to outcomes; treatments to readmission or cost.
   - Directly address CROSS-SHEET HYPOTHESIS in conclusion.
   - Explicitly flag data that contradicts the hypothesis.

5. OUTPUT FORMAT:
   - Use professional Markdown.
   - Every claim MUST cite a specific metric from the JSON.

6. SEMANTIC NULLS PROTOCOL — Apply BEFORE any null-rate decision:
   CRITICAL: Not all clinical nulls are data failures. Classify every high-null column:
   a. TECHNICAL GAPS: Missing mandatory clinical fields (e.g., Age, Diagnosis Code, Admission Date, Insurance ID). If >5% null → flag prominently as a data gap.
   b. MEANINGFUL SPARSITY: Columns representing rare clinical events (e.g., Readmission, Complication, Adverse Event, ICU Transfer, Mortality). Nulls = valid "Event Did Not Occur" state.
   Rule: For sparse clinical event columns, treat null as baseline (no event). Analyze non-null rows as the "High-Risk Cohort". Do NOT add a "Data Limitations" subsection for these columns. Still report N for all subgroups.

GOLD STANDARD HEALTHCARE ANALYTICS FRAMEWORKS — Apply ALL three + two healthcare-specific lenses:

A. PARETO CONCENTRATION (80/20 Rule):
   - Which 20% of diagnoses, conditions, or patient cohorts drive 80% of cost, readmissions, or ER visits?
   - Identify high-utilizer patients (top decile of visit frequency or cost).
   - Flag: if top 10% of patients account for >60% of total cost, this is a high-risk cohort signal.

B. BIMODAL DISTRIBUTION — Acute vs. Chronic Segmentation:
   - Are length-of-stay, cost, or readmission distributions bimodal?
   - Is the patient population split between acute/episodic cases and chronic/recurring ones?
   - Report both cluster means, patient share, and cost share.
   - NEVER average acute and chronic patients into a single headline metric.

C. PARADOX DETECTION — Counter-Intuitive Clinical Findings:
   - Tag confirmed paradoxes with [PARADOX].
   - Classic healthcare paradoxes: higher staffing ratio → higher readmission; more procedures → worse outcomes;
     premium insurance tier → higher ER utilization; shorter LOS → higher 30-day readmission.

D. HEALTHCARE-SPECIFIC: Readmission & Outcome Risk Stratification:
   - Identify the strongest predictors of readmission or adverse outcomes from the available columns.
   - Segment patients by risk tier (Low / Medium / High) if enough numeric data is present.
   - Flag any demographic or socioeconomic proxies that appear as outcome predictors (equity signal).

E. HEALTHCARE-SPECIFIC: Cost & Efficiency Benchmarking:
   - Compare cost per encounter or cost per diagnosis across provider types, departments, or facilities.
   - Identify outlier providers (>2 std above mean cost with no corresponding outcome improvement).
   - Flag zero-outcome high-cost procedures as candidates for clinical review.
