ANALYSIS RULES:
1. MODE ADAPTATION:
   - "joined": Cross-entity data (players + teams, matches + stats) is the primary source.
   - "both": Start team/league macro view, then drill into individual performance.
   - "separate": Find patterns or contradictions across seasons, positions, or leagues.

2. DATA INTEGRITY:
   - NO IDs: Never use player IDs, match IDs as analytical dimensions.
   - DISCRETE: Treat position, formation, league tier, and outcome codes as categorical.
   - SIGNIFICANCE: Correlations < |0.10| = "Noise". Small sample sizes (N<30) must be flagged.
   - QUALITY: Apply SEMANTIC NULLS PROTOCOL (rule 6) — not all nulls are data failures.
   - CAUSATION: Use "associated with" not "caused" unless correlation > 0.6.

3. DATA HONESTY: No speculation. If corr <0.1, state "inconclusive".
   Small sample caveats MUST be included. If N<30 for any segment, flag as "low confidence".
   For sparse achievement columns (e.g., Medal, Award), analyze the non-null "Elite Tier" — do NOT add a limitations subsection.

4. STRUCTURE:
   - Link individual metrics to team outcomes and league standings.
   - Directly address CROSS-SHEET HYPOTHESIS in conclusion.
   - Explicitly flag data that contradicts the hypothesis.

5. OUTPUT FORMAT:
   - Use professional Markdown.
   - Every claim MUST cite a specific metric from the JSON.

6. SEMANTIC NULLS PROTOCOL — Apply BEFORE any null-rate decision:
   CRITICAL: Do not treat all nulls as data failures. Classify every high-null column:
   a. TECHNICAL GAPS: Missing mandatory fields (e.g., Age, Birth Date, Height, Weight). If >5% null → report as a Data Limitation.
   b. MEANINGFUL SPARSITY: Columns representing rare achievements (e.g., Medal, Award, Trophy, Title, Record). Nulls = valid "Did Not Achieve" state.
   Rule: For sparse achievement columns, treat null as baseline (no achievement). Analyze the non-null rows as the "Elite Tier" or "Target Cohort". Do NOT add a "Data Limitations" subsection for these columns.

GOLD STANDARD SPORTS ANALYTICS FRAMEWORKS — Apply ALL three + two sports-specific lenses:

A. PARETO CONCENTRATION (80/20 Rule):
   - Which 20% of players, teams, or positions account for 80% of goals, wins, or points?
   - Quantify star-player dependency: if top-3 players account for >60% of output, flag as fragility risk.
   - Identify "carrying" entities vs. average contributors.

B. BIMODAL DISTRIBUTION — Elite vs. Pack Separation:
   - Is performance data bimodal? (Elite performers vs. average field?)
   - Are game outcomes bimodal? (Dominant wins + dominant losses with few close games?)
   - Report both cluster means. Label clusters as "Elite Tier" and "Field Tier".
   - Never report a single average across bimodal performance data.

C. PARADOX DETECTION — Counter-Intuitive Sports Findings:
   - Tag confirmed paradoxes with [PARADOX].
   - Classic sports paradoxes: higher possession % → fewer goals; more shots → lower conversion rate;
     highest-paid players ranked lowest in key performance metrics; home advantage absent or reversed.

D. SPORTS-SPECIFIC: Form & Momentum Analysis:
   - If time/sequence data is available, check for performance streaks vs. regression to mean.
   - Identify if recent form (last N matches) diverges significantly from season average.
   - Flag "false peaks": short hot streaks inflating season stats.

E. SPORTS-SPECIFIC: Positional & Tactical Efficiency:
   - Compare performance metrics across positions or tactical formations.
   - Identify the most and least efficient positions per unit of playing time or cost.
   - Flag positions where performance variance is highest (high risk, high reward roles).
